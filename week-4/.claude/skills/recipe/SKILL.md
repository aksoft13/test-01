# /recipe — 냉장고 재료 기반 레시피 생성기

## 실행 즉시 수행할 작업

1. `3.myRecipeSkill/ingredients/` 디렉토리의 **모든 JSON 파일을 읽으세요** (Glob → Read)
2. 재료 목록을 파악하고 오늘 날짜(currentDate)와 비교해 **유통기한 임박 재료(3일 이내)를 식별**하세요
3. 레시피를 결정하세요:
   - `/recipe` (인수 없음) → 보유 재료로 만들 수 있는 요리 **3가지를 제안**하고 사용자가 선택하게 하세요. 유통기한 임박 재료를 우선 활용하세요.
   - `/recipe 김치볶음밥` → 해당 요리를 바로 생성하세요
4. 선택된 레시피에 대해 **마크다운 파일 + 썸네일 이미지를 생성**하세요
5. `3.myRecipeSkill/recipes/` 폴더에 저장하세요

---

## 재료 JSON 파일 형식

```json
{
  "name": "계란",
  "name_en": "egg",
  "quantity": 5,
  "unit": "개",
  "category": "단백질",
  "expiry": "2026-04-25",
  "storage": "냉장",
  "notes": "",
  "thumbnail": "./thumbnails/egg.png"
}
```

**기본 양념은 항상 보유한다고 가정:** 간장, 설탕, 고추장, 식용유, 소금, 후추, 참기름, 다진마늘

## 레시피 마크다운 저장 형식

파일 위치: `3.myRecipeSkill/recipes/{recipe-name}.md`
썸네일 위치: `3.myRecipeSkill/recipes/thumbnails/{recipe-name}.png`

```markdown
![thumbnail](./thumbnails/{recipe-name}.png)

# {레시피 이름}

> ⏱️ 조리시간: {X}분 | 🍽️ {인분} | 난이도: ⭐ 쉬움

## 📝 재료
- {재료 1} — {양}
- {재료 2} — {양}

## 👨‍🍳 만드는 법
1. {단계 1}
2. {단계 2}

## 💡 꿀팁
- {효율적인 조리 팁}
- {설거지 최소화 팁}
- {재료 대체 가능 옵션}

---
*냉장고 재료: {사용된 재료 목록}*
*생성일: {오늘 날짜}*
```

## 레시피 생성 규칙

### 기본 조건 (항상 적용)
- **1인분** 기준으로 작성
- 조리 시간 **15분 이하** 필수
- 난이도: **자취생 수준** — 요리 경험이 거의 없어도 성공 가능해야 함
  - 칼질 최소화 (찢기, 가위 사용 허용)
  - 불 조절 단순하게 (중불 고정 등)
  - 단계별 설명은 초보자도 이해할 수 있게 구체적으로

### 장비 조건
- **특수 장비 불필요** (가스레인지/인덕션 + 프라이팬/냄비만으로 완성)
- 오븐, 에어프라이어는 선택 사항으로만 언급

### 실용성 조건
- 설거지를 줄이는 **실용적인 팁** 포함 (한 냄비 조리, 키친타올 활용 등)
- 가능한 경우 **재료 대체안** 제시
- 실패 포인트가 있다면 **미리 경고**

### 말투
- 친절하고 격려하는 스타일 ("쉬워요!", "누구나 성공해요!", "실패해도 맛있어요!")

## 썸네일 이미지 생성

`fal-image-gen` 스킬을 사용하여 완성된 요리 사진을 생성하세요.
프롬프트 형식: `"{요리 이름}, Korean home cooking, top-down food photography, warm lighting, appetizing, simple ingredients, 15-minute meal"`

SSL 인증서 문제 해결을 위해 환경변수를 설정하고 실행하세요:
```bash
SSL_CERT_FILE=$(python3 -c "import certifi; print(certifi.where())") \
python3 .claude/skills/fal-image-gen/scripts/generate.py \
  -p "{프롬프트}" -o {출력경로} -W 512 -H 512 --steps 3
```

## 재료 관리 (선택적)

`/recipe` 실행 중 또는 별도로 재료를 관리할 수 있습니다:

| 명령 | 동작 |
|------|------|
| `재료 추가: 우유 200ml 냉장 2026-04-30` | `milk.json` 파일 생성 |
| `재료 수정: 계란 3개로 변경` | `egg.json`의 quantity 수정 |
| `재료 삭제: 두부 다 썼어` | `tofu.json` 삭제 |
| `냉장고 확인` | 모든 재료를 카테고리별로 표시 |
